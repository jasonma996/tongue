## 🎨 设计特点

### 视觉风格
- ✨ Glassmorphism（玻璃态）设计
- 🎭 渐变动画背景
- 🌈 现代配色系统
- 📱 完全响应式布局
- ⚡ 流畅的交互动画

### 技术栈
- **后端**: Flask (Python)
- **前端**: HTML5 + CSS3 + JavaScript
- **字体**: Inter (Google Fonts)
- **动画**: Lottie + GSAP
- **AI引擎**: 智谱AI GLM-4V-Flash（免费）+ 规则引擎fallback
